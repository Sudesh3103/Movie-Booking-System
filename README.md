🎬 Movie Booking System
A robust, web-based application designed to streamline the movie ticket reservation process. Built using Java, JSP, and Servlets, this project demonstrates a full-stack approach to managing cinema bookings, user accounts, and show schedules.

🚀 Features
User Authentication: Secure login and registration for customers.

Movie Catalog: Browse current and upcoming movies with details.

Real-time Booking: Select shows, choose seats, and book tickets instantly.

Admin Dashboard: Specialized access to add/remove movies, manage show timings, and view booking history.

Responsive UI: Clean and intuitive interface built with HTML5, CSS3, and JavaScript.

🛠️ Tech Stack
Backend: Java (JSP & Servlets)

Frontend: HTML5, CSS3, JavaScript, Bootstrap

Database: MySQL (JDBC for connectivity)

Server: Apache Tomcat 9.0+
